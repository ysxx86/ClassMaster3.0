## 快速开始

### 1. 启动服务器

```bash
python3 server.py
```

服务器将在 http://localhost:5000 启动

### 2. 登录系统

使用管理员账号登录系统

### 3. 访问班主任考核模块

在左侧导航栏中点击 **"班主任考核"** 选项卡

## 首次使用配置

### 步骤1: 设置教师类型（管理员）

1. 点击 **"教师管理"** 标签
2. 为每位教师选择类型：
   - 正班主任
   - 副班主任
3. 点击 **"保存"** 按钮

### 步骤2: 配置评分人员（管理员）

1. 点击 **"评分人员"** 标签
2. 点击 **"添加评分人员"** 按钮
3. 选择用户并设置评分权限：
   - 可以指定特定项目
   - 或者授权评所有项目
4. 保存设置

### 步骤3: 开始评分

1. 点击 **"评分录入"** 标签
2. 选择学期（如：2024-2025-1）
3. 选择要评分的教师
4. 在各个考核项目中输入分数（0-100）
5. 点击 **"保存"** 按钮

### 步骤4: 计算结果（管理员）

1. 点击 **"考核结果"** 标签
2. 选择学期
3. 点击 **"计算结果"** 按钮
4. 系统自动计算并显示排名

## 评分规则

1. **评分范围**: 0-100分
2. **多人评分**: 每个项目可以由多人评分
3. **去极值**: 当某项目有3个或以上评分时，自动去掉最高分和最低分
4. **平均计算**: 剩余评分取平均值
5. **加权汇总**: 总分 = Σ(各项目平均分 × 权重)

## 常见问题

### Q1: 如何修改考核项目？
**A**: 管理员登录后，进入"考核项目"标签，可以添加、编辑或删除项目。

### Q2: 评分后可以修改吗？
**A**: 可以。评分人可以重新提交评分，系统会覆盖之前的评分。

### Q3: 如何查看历史学期的考核结果？
**A**: 在"考核结果"标签中选择对应的学期即可。

### Q4: 权重可以调整吗？
**A**: 可以。管理员在"考核项目"标签中编辑项目时可以调整权重。

### Q5: 如何导出考核结果？
**A**: 目前版本暂不支持导出，后续版本会添加Excel和PDF导出功能。

## 技术支持

如遇到问题，请检查：

1. **数据库连接**: 确保 students.db 文件存在且可访问
2. **权限设置**: 确认用户有相应的操作权限
3. **浏览器兼容**: 建议使用Chrome、Firefox或Edge浏览器
4. **日志查看**: 查看 logs/server.log 了解详细错误信息

## 数据备份建议

定期备份 students.db 文件，特别是在：
- 计算考核结果之前
- 修改考核项目之前
- 学期结束时

备份命令：
```bash
cp students.db students.db.backup_$(date +%Y%m%d_%H%M%S)
```

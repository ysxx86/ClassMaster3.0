# 学科班级分配功能优化说明

## 修改内容

### 1. 班级选择方式改为复选框
- 原来：使用 `<select multiple>` 多选下拉框，需要按住 Ctrl/Cmd 键
- 现在：使用复选框列表，可以直接勾选多个班级
- 优点：更直观，更易用

### 2. 修改说明文字
- **角色分配**：保持不变，所有教师都可以分配角色
- **班级分配**：说明主要用于正班主任和副班主任
- **学科班级分配**：说明正班主任无需手动分配

### 3. 优化显示逻辑
- 正班主任选择后，隐藏班级选择区域，显示提示信息
- 其他角色选择后，显示班级复选框列表
- 添加调试日志，方便排查问题

### 4. 改进当前分配显示
- 添加表格边框，更清晰
- 添加调试日志输出
- 优化表格样式

## 使用方法

### 为科任老师分配班级

1. 进入"教师分配"页面
2. 点击"学科班级分配"标签页
3. 选择教师（例如：张老师）
4. 选择学科（例如：数学）
5. 勾选要分配的班级（可以勾选多个）
6. 点击"分配选中的班级"按钮

### 查看当前分配

选择教师后，下方会自动显示该教师的所有分配情况，按学科分组显示。

### 移除分配

点击对应学科行的"移除"按钮，可以删除该学科的所有班级分配。

## 排查问题

如果"当前分配"显示为空：

1. 打开浏览器开发者工具（F12）
2. 切换到"控制台"标签
3. 选择教师，查看控制台输出：
   - `加载教师分配情况，教师ID: xxx`
   - `API返回数据: {...}`
   - `渲染分配情况，数据条数: x`
   - `按学科分组后: {...}`

4. 如果API返回数据为空，检查数据库：
   ```bash
   sqlite3 students.db "SELECT * FROM teacher_classes WHERE teacher_id='教师ID';"
   ```

5. 如果数据库有数据但API返回为空，检查服务器日志

# 教师导入功能重构 - 完成文档

## 概述
将"导入班主任"功能重构为"导入教师名单"，支持导入所有类型的教师（班主任、科任老师、行政、校级领导等），并支持新增和更新现有用户。

## 需求变更

### 原需求
- 功能名称：导入班主任
- 必填字段：用户名、班级
- 功能：仅新增班主任账户

### 新需求
- 功能名称：导入教师名单
- 必填字段：仅用户名
- 可选字段：年级、班级、角色、任教学科
- 功能：支持新增和更新教师账户
- 预览功能：绿色标记新增用户，红色标记已存在用户（将更新）

## 实现细节

### 1. 后端API（users.py）

#### 预览导入API：`/api/users/preview-import`
- 方法：POST
- 功能：预览Excel文件中的教师数据，不执行真正的导入
- 输入：Excel文件（multipart/form-data）
- 输出：
  ```json
  {
    "status": "ok",
    "message": "找到 X 条教师数据，其中 Y 条新增，Z 条更新，W 条无效",
    "preview": [
      {
        "row": 2,
        "username": "张老师",
        "grade": "一年级",
        "class_name": "一年级1班",
        "role": "正班主任",
        "subjects": ["语文", "数学"],
        "status": "新增",  // 或 "更新" 或 "无效"
        "reason": "",
        "is_valid": true,
        "is_existing": false
      }
    ],
    "stats": {
      "total": 10,
      "new": 5,
      "update": 3,
      "invalid": 2
    },
    "file_path": "uploads/20260117_teachers.xlsx"
  }
  ```

#### 确认导入API：`/api/users/confirm-import`
- 方法：POST
- 功能：确认导入预览过的教师数据，支持新增和更新
- 输入：
  ```json
  {
    "file_path": "uploads/20260117_teachers.xlsx"
  }
  ```
- 输出：
  ```json
  {
    "status": "ok",
    "message": "成功新增 X 个教师账户，更新 Y 个教师账户，跳过 Z 个",
    "added_count": 5,
    "updated_count": 3,
    "skipped_count": 2,
    "added": [...],
    "updated": [...],
    "skipped": [...],
    "passwords": [
      {
        "username": "张老师",
        "class_name": "一年级1班",
        "password": "abc123"
      }
    ]
  }
  ```

#### Excel模板生成
- 函数：`create_teacher_import_template()`
- 模板列：
  1. 用户名（必填）
  2. 年级（可选）
  3. 班级（可选）
  4. 角色（可选：正班主任/副班主任/科任老师/行政/校级领导）
  5. 任教学科（可选，多个学科用逗号分隔）

## 数据验证规则

### 必填字段
- 用户名：不能为空

### 可选字段验证
1. 班级：如果提供，必须在系统中已存在
2. 角色：如果提供，必须是以下之一：
   - 正班主任
   - 副班主任
   - 科任老师
   - 行政
   - 校级领导
3. 任教学科：如果提供，所有学科必须在系统中已存在

### 导入逻辑
1. 用户名已存在 → 更新用户信息（角色、班级、任教学科）
2. 用户名不存在 → 新增用户，生成随机密码
3. 数据验证失败 → 跳过该记录

## 数据库操作

### 新增用户
```sql
INSERT INTO users (id, username, password_hash, is_admin, primary_role, class_id, reset_password, created_at, updated_at) 
VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?)
```

### 更新用户
```sql
UPDATE users 
SET primary_role = ?, class_id = ?, updated_at = ?
WHERE id = ?
```

### 任教学科管理
```sql
-- 删除旧的任教学科
DELETE FROM teacher_subjects WHERE teacher_id = ?

-- 添加新的任教学科
INSERT INTO teacher_subjects (teacher_id, subject_id)
VALUES (?, ?)
```

## 使用说明

### 导入步骤
1. 点击"Excel导入教师名单"按钮
2. 选择Excel文件（或下载模板填写）
3. 系统自动预览数据：
   - 绿色：新增用户
   - 红色：已存在用户（将更新）
   - 灰色：无效数据
4. 检查预览结果，确认无误后点击"确认导入"
5. 查看导入结果：
   - 新增用户：显示生成的密码
   - 更新用户：显示更新的信息
   - 跳过记录：显示失败原因
6. 复制密码信息（如有新增用户）
7. 关闭窗口，用户列表自动刷新

### Excel模板格式
| 用户名 | 年级 | 班级 | 角色 | 任教学科 |
|--------|------|------|------|----------|
| 张老师 | 一年级 | 一年级1班 | 正班主任 | 语文,数学 |
| 李老师 | 二年级 | 二年级3班 | 副班主任 | 英语 |
| 王老师 | 三年级 |  | 科任老师 | 体育,音乐 |

注意：
- 用户名为必填项
- 其他字段均为可选
- 任教学科多个用逗号分隔
- 班级需先在系统中创建

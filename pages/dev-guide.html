项目架构

ClassMaster 采用经典的三层架构设计，各层职责清晰分离：

前端展示层
HTML + CSS + JavaScript（原生，无框架依赖）
后端业务层
Python + Flask（轻量级 Web 框架）
数据存储层
SQLite（免配置本地数据库）

项目目录结构

ClassMaster3.0/

编码规范

Python 后端

代码风格
遵循 PEP 8
类名
PascalCase（大驼峰）
函数/变量名
snake_case（下划线）
常量
UPPER_SNAKE_CASE
文档字符串
公共函数必须添加 Docstring
行宽限制
最大 120 字符

前端

HTML
语义化标签，2 空格缩进
CSS 类名
BEM 风格，kebab-case
JS 变量/函数
camelCase（小驼峰）
JS 类
PascalCase（大驼峰）
变量声明
使用 const/let，禁止 var

开发流程

1

Fork 仓库

在 Gitee/GitHub 上 Fork 项目仓库

2

克隆到本地

git clone https://gitee.com/你的用户名/class-master.git
3

创建分支

git checkout -b feature/你的功能名
4

开发功能

按照编码规范进行开发

5

本地测试

python server.py

启动并测试功能

6

提交代码

git add . && git commit -m "描述"
7

推送分支

git push origin feature/你的功能名
8

创建 PR

在 Gitee/GitHub 上提交 Pull Request

9

代码审查

等待项目维护者审查代码

10

合并上线

审查通过后合并到主分支

申请 DeepSeek API Key

项目克隆

Gitee（国内推荐）

git clone https://gitee.com/ysxx86/class-master.git
cd class-master

GitHub

git clone https://github.com/ysxx86/ClassMaster3.0.git
cd ClassMaster3.0

配置文件

创建 .env 文件

cp .env.example .env

编辑 .env 文件

添加 DeepSeek API Key：

DEEPSEEK_API_KEY=sk-your-api-key-here
config.json 在首次运行时自动生成，使用默认设置即可

依赖安装

pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
项目内置依赖自动检测机制（dependency_manager.py），首次启动时会自动检查并提示缺失依赖

数据库初始化

SQLite 数据库在首次启动时自动创建和初始化，无需手动操作。

默认管理员账号：用户名 admin，密码 admin123。首次登录后请立即修改密码！

本地启动

python server.py
# 或
python app.py

启动后访问：http://localhost:8080

可选启动参数

python server.py --host 0.0.0.0 --port 8080 --debug

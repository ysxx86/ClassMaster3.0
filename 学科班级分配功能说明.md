# 学科班级分配功能说明

## 功能概述

新增了"学科班级分配"功能，允许管理员为教师分配任教的班级和学科。解决了科任老师、副班主任等可能任教多个班级的问题。

## 权限逻辑

### 1. 正班主任
- 只有一个班级（class_id字段）
- 只任教自己的班级
- 可以编辑本班级所有学科的成绩
- 成绩管理只显示自己的班级
- **不需要在"学科班级分配"中手动分配**

### 2. 副班主任/科任老师/行政/校级领导
- 可能任教多个班级的多个学科
- 需要在"学科班级分配"中手动分配
- 成绩管理只显示自己任教的班级
- 只能编辑自己任教的学科

### 3. 超级管理员
- 可以查看和编辑所有班级的所有学科

## 数据库变更

### 新增表：teacher_classes

```sql
CREATE TABLE teacher_classes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    teacher_id TEXT NOT NULL,
    class_id INTEGER NOT NULL,
    subject_id INTEGER NOT NULL,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (class_id) REFERENCES classes(id) ON DELETE CASCADE,
    FOREIGN KEY (subject_id) REFERENCES subjects(id) ON DELETE CASCADE,
    UNIQUE(teacher_id, class_id, subject_id)
)
```

**说明**：
- `teacher_id`：教师ID
- `class_id`：班级ID
- `subject_id`：学科ID
- 唯一约束：同一个教师不能重复分配到同一个班级的同一个学科

## 使用方法

### 1. 进入教师分配页面

点击左侧菜单"管理后台" → "教师分配"

### 2. 切换到"学科班级分配"标签页

页面有三个标签页：
- 角色分配
- 班级分配
- **学科班级分配**（新增）

### 3. 分配步骤

1. **选择教师**：从下拉框中选择要分配的教师
2. **选择学科**：选择该教师任教的学科
3. **选择班级**：按住 Ctrl 或 Cmd 键可多选班级
4. **点击"分配"按钮**：完成分配

### 4. 查看和管理分配

- 选择教师后，下方会显示该教师当前的所有分配
- 按学科分组显示任教的班级
- 可以点击"移除"按钮删除某个学科的所有班级分配

## 示例场景

### 场景1：科任老师任教多个班级

张老师是科任老师，任教1年级1班和1年级2班的数学：

1. 选择教师：张老师
2. 选择学科：数学
3. 选择班级：按住 Ctrl，同时选中"1年级1班"和"1年级2班"
4. 点击"分配"

完成后，张老师在成绩管理页面可以看到这两个班级的学生，但只能编辑数学成绩。

### 场景2：副班主任任教多个学科

李老师是1年级1班的副班主任，同时任教1年级1班的道法和1年级2班的道法：

**第一次分配**：
1. 选择教师：李老师
2. 选择学科：道法
3. 选择班级：选中"1年级1班"和"1年级2班"
4. 点击"分配"

如果李老师还任教其他学科，重复上述步骤即可。

### 场景3：正班主任

王老师是1年级1班的正班主任：

- **不需要在"学科班级分配"中分配**
- 系统会自动根据 class_id 字段判断
- 王老师可以编辑1年级1班所有学科的成绩

## 新增API接口

### 1. 获取教师任教的班级列表
```
GET /api/teacher-classes/<teacher_id>
```

### 2. 分配教师到班级和学科
```
POST /api/teacher-classes
Body: {
    "teacher_id": "教师ID",
    "subject_id": "学科ID",
    "class_ids": ["班级ID1", "班级ID2", ...]
}
```

### 3. 移除教师的班级分配
```
DELETE /api/teacher-classes/<assignment_id>
```

### 4. 获取某个班级某个学科的任教教师
```
GET /api/class-subject-teachers/<class_id>/<subject_id>
```

## 注意事项

1. **正班主任不需要手动分配**：系统会自动根据 class_id 判断
2. **必须先分配学科**：在"学科分配"功能中先给教师分配学科，然后才能在"学科班级分配"中分配班级
3. **可以多选班级**：按住 Ctrl 或 Cmd 键可以同时选择多个班级
4. **重复分配会覆盖**：如果重新分配同一个学科，会先删除旧的班级分配，然后添加新的
5. **级联删除**：删除教师、班级或学科时，相关的分配记录会自动删除

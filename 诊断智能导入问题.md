# 诊断智能导入问题

## 问题描述
选择文件后没有预览，确认导入按钮也无法使用。

## 可能的原因

### 1. JavaScript文件加载顺序问题
**检查方法：**
打开浏览器开发者工具（F12），查看Console标签，应该看到：
```
smart-grade-import.js 开始加载...
smart-grade-import.js 加载完成，准备定义函数...
智能导入函数已导出到window对象
- initSmartGradeImport: function
- previewSmartImport: function
- confirmSmartImport: function
- resetSmartImportModal: function
```

**解决方法：**
确保在pages/grades.html中，smart-grade-import.js在grades.js之前加载：
```html
<script src="/js/libs/bootstrap.bundle.min.js"></script>
<script src="/js/data.js"></script>
<script src="/js/smart-grade-import.js"></script>  <!-- 必须在grades.js之前 -->
<script src="/js/grades.js"></script>
```

### 2. 元素ID不匹配
**检查方法：**
在Console中输入：
```javascript
document.getElementById('gradeFile')
document.getElementById('previewContent')
document.getElementById('confirmImportGrades')
```
应该都返回对应的元素，而不是null。

**解决方法：**
如果返回null，检查HTML中的元素ID是否正确。

### 3. 事件监听器未绑定
**检查方法：**
在Console中输入：
```javascript
typeof initSmartGradeImport
```
应该返回 "function"

然后手动调用：
```javascript
initSmartGradeImport()
```
应该看到：
```
初始化智能成绩导入...
绑定文件选择事件
绑定拖放事件
绑定下载模板按钮
绑定确认导入按钮
绑定模态框关闭事件
智能成绩导入初始化完成
```

### 4. 文件选择事件未触发
**检查方法：**
选择文件后，在Console中应该看到：
```
文件选择事件触发
选择的文件: xxx.xlsx
文件已选择: xxx.xlsx
文件类型验证通过，准备预览...
触发预览...
预览导入 - 学期: 2025-2026学年第一学期
```

**解决方法：**
如果没有看到这些日志，说明事件监听器没有正确绑定。

### 5. API请求失败
**检查方法：**
在Network标签中查看是否有 `/api/grades/preview-import` 请求。

**解决方法：**
- 检查服务器是否正常运行
- 检查API路由是否正确
- 查看服务器日志

## 快速诊断步骤

### 步骤1：打开测试页面
访问：`http://localhost:5000/test_smart_import_ui.html`

这个页面会显示所有的控制台日志，方便诊断。

### 步骤2：检查JavaScript加载
在浏览器Console中输入：
```javascript
console.log('检查函数:', {
    initSmartGradeImport: typeof initSmartGradeImport,
    previewSmartImport: typeof previewSmartImport,
    confirmSmartImport: typeof confirmSmartImport
});
```

应该看到所有函数都是 "function"。

### 步骤3：手动触发初始化
```javascript
initSmartGradeImport();
```

### 步骤4：选择文件并观察日志
选择一个Excel文件，观察Console中的日志输出。

### 步骤5：检查预览容器
```javascript
console.log('预览容器:', document.getElementById('previewContent'));
console.log('确认按钮:', document.getElementById('confirmImportGrades'));
```

## 临时解决方案

如果智能导入无法工作，可以暂时使用旧版导入：

1. 在grades.js中，将初始化代码改为：
```javascript
// 强制使用旧版导入
initGradesImport();
```

2. 或者在Console中手动调用旧版导入：
```javascript
initGradesImport();
```

## 完整的调试代码

在Console中运行以下代码进行完整诊断：

```javascript
console.log('=== 智能导入诊断 ===');

// 1. 检查函数是否存在
console.log('1. 函数检查:');
console.log('  initSmartGradeImport:', typeof initSmartGradeImport);
console.log('  previewSmartImport:', typeof previewSmartImport);
console.log('  confirmSmartImport:', typeof confirmSmartImport);

// 2. 检查DOM元素
console.log('2. DOM元素检查:');
console.log('  gradeFile:', document.getElementById('gradeFile'));
console.log('  previewContent:', document.getElementById('previewContent'));
console.log('  confirmImportGrades:', document.getElementById('confirmImportGrades'));
console.log('  importSemester:', document.getElementById('importSemester'));

// 3. 检查事件监听器
const fileInput = document.getElementById('gradeFile');
if (fileInput) {
    console.log('3. 文件输入框存在');
    console.log('  事件监听器数量:', getEventListeners(fileInput).change?.length || 0);
} else {
    console.log('3. 文件输入框不存在！');
}

// 4. 尝试手动初始化
console.log('4. 尝试手动初始化...');
try {
    if (typeof initSmartGradeImport === 'function') {
        initSmartGradeImport();
        console.log('  初始化成功');
    } else {
        console.log('  initSmartGradeImport 不是函数');
    }
} catch (e) {
    console.error('  初始化失败:', e);
}

console.log('=== 诊断完成 ===');
```

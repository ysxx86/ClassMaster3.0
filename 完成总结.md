## 功能特点

### 1. 灵活的字段要求
- **必填**：仅用户名
- **可选**：年级、班级、角色、任教学科

### 2. 智能识别
- 自动识别新增用户（绿色）
- 自动识别更新用户（红色）
- 自动识别无效数据（灰色）

### 3. 双重功能
- **新增用户**：创建账户 + 生成密码
- **更新用户**：更新信息 + 保留密码

### 4. 完善的验证
- 用户名不能为空
- 班级必须存在
- 角色必须有效
- 学科必须存在

### 5. 友好的提示
- 详细的错误原因
- 清晰的统计信息
- 分类的结果展示

## 使用流程

```
1. 准备Excel文件
   ↓
2. 上传并预览
   ↓
3. 检查颜色标记
   - 绿色：新增
   - 红色：更新
   - 灰色：无效
   ↓
4. 确认导入
   ↓
5. 查看结果
   - 新增用户密码
   - 更新用户信息
   - 跳过记录原因
   ↓
6. 复制密码（如有）
   ↓
7. 完成
```

## 常见问题快速解答

| 问题 | 答案 |
|------|------|
| 只有用户名可以导入吗？ | 可以，其他都是可选的 |
| 更新用户会改密码吗？ | 不会，只更新角色、班级、学科 |
| 班级不存在怎么办？ | 先在班级管理中创建 |
| 学科不存在怎么办？ | 先在学科管理中创建 |
| 密码在哪里看？ | 导入结果窗口（仅显示一次） |
| 如何重置密码？ | 用户列表中点击"重置密码" |

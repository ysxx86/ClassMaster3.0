## ⚡ 快速集成（2步）

### 步骤1：添加CSS引用

在页面的 `<head>` 标签中，找到其他CSS引用的位置，添加：

```html
<!-- 更新通知样式 -->
<link rel="stylesheet" href="/css/update-notification.css">
```

**完整示例：**
```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
    
    <!-- Bootstrap CSS -->
    <link href="/css/bootstrap.min.css" rel="stylesheet">
    
    <!-- 图标库 -->
    <link href="/css/boxicons-local.css" rel="stylesheet">
    
    <!-- 更新通知样式 -->
    <link rel="stylesheet" href="/css/update-notification.css">
    
    <!-- 页面自己的样式 -->
    <link rel="stylesheet" href="/css/xxx.css">
</head>
```

### 步骤2：添加JS引用

在页面底部 `</body>` 标签前，找到其他JS引用的位置，添加：

```html
<!-- 更新通知脚本 -->
<script src="/js/update-notification.js"></script>
```

**完整示例：**
```html
    <!-- Bootstrap JS -->
    <script src="/js/libs/bootstrap.bundle.min.js"></script>
    
    <!-- 更新通知脚本 -->
    <script src="/js/update-notification.js"></script>
    
    <!-- 页面自己的脚本 -->
    <script src="/js/xxx.js"></script>
</body>
</html>
```

## ⚠️ 注意事项

1. **CSS位置**：放在 `<head>` 中，在Bootstrap之后
2. **JS位置**：放在 `</body>` 前，在Bootstrap JS之后
3. **路径**：确保路径正确（相对于页面位置）
4. **顺序**：必须在Bootstrap之后加载

## 🔍 路径说明

### 如果页面在根目录
```html
<link rel="stylesheet" href="css/update-notification.css">
<script src="js/update-notification.js"></script>
```

### 如果页面在 pages/ 目录
```html
<link rel="stylesheet" href="/css/update-notification.css">
<script src="/js/update-notification.js"></script>
```

或者：
```html
<link rel="stylesheet" href="../css/update-notification.css">
<script src="../js/update-notification.js"></script>
```

## ✅ 验证方法

集成后，打开浏览器开发者工具（F12）：

1. **检查CSS加载**：
   - 打开 Network 标签
   - 刷新页面
   - 查找 `update-notification.css`
   - 状态应该是 200

2. **检查JS加载**：
   - 打开 Network 标签
   - 刷新页面
   - 查找 `update-notification.js`
   - 状态应该是 200

3. **检查功能**：
   - 右上角应该显示紫色圆形按钮
   - 在控制台执行：`showUpdateModal()`
   - 应该弹出更新通知模态框

## 🐛 常见问题

### 问题1：右上角图标不显示

**可能原因：**
- CSS文件路径错误
- CSS文件未加载
- 被其他元素遮挡

**解决方法：**
```javascript
// 在控制台检查
document.getElementById('updateIconContainer')
// 如果返回 null，说明JS未正确执行
```

### 问题2：点击图标没反应

**可能原因：**
- JS文件路径错误
- JS文件未加载
- Bootstrap未加载

**解决方法：**
```javascript
// 在控制台检查
typeof showUpdateModal
// 应该返回 "function"
```

### 问题3：模态框样式错乱

**可能原因：**
- Bootstrap CSS未加载
- CSS加载顺序错误

**解决方法：**
- 确保Bootstrap CSS在前
- 确保update-notification.css在后

## 🚀 一键集成命令

如果你想一次性集成到所有页面，可以使用以下Python脚本：

```python
import os
import re

# 需要集成的页面列表
pages = [
    'pages/students.html',
    'pages/grades.html',
    'pages/comments.html',
    'pages/performance.html',
    'pages/export.html',
    'pages/admin.html',
    'pages/teacher_assignment.html',
    'pages/subjects.html',
    'pages/settings.html'
]

css_line = '    <link rel="stylesheet" href="/css/update-notification.css">\n'
js_line = '    <script src="/js/update-notification.js"></script>\n'

for page in pages:
    if not os.path.exists(page):
        print(f"跳过：{page} (文件不存在)")
        continue
    
    with open(page, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 检查是否已经集成
    if 'update-notification' in content:
        print(f"跳过：{page} (已集成)")
        continue
    
    # 添加CSS（在</head>之前）
    if '</head>' in content and css_line not in content:
        content = content.replace('</head>', css_line + '</head>')
    
    # 添加JS（在</body>之前）
    if '</body>' in content and js_line not in content:
        content = content.replace('</body>', js_line + '</body>')
    
    # 写回文件
    with open(page, 'w', encoding='utf-8') as f:
        f.write(content)
    
    print(f"✅ 完成：{page}")

print("\n集成完成！")
```

保存为 `integrate_update_notification.py`，然后运行：
```bash
python integrate_update_notification.py
```

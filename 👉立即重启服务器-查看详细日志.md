### 方法2：手动操作
1. **停止当前服务器**
   - 找到运行 `python server.py` 的命令行窗口
   - 按 `Ctrl + C` 停止服务器

2. **重新启动服务器**
   ```bash
   python server.py
   ```

3. **打开新的命令行窗口查看日志**
   ```bash
   powershell -Command "Get-Content logs\root_server.log -Wait -Tail 50"
   ```

1. **学生匹配是否成功**

2. **检测到的学科列表**
   - `detected_subjects: ['道法', '语文', ...]` 显示识别的学科

   - `class_id=11 (type=<class 'int'>)` 显示类型

4. **teaching_map 的键**
   - `teaching_map keys: ['11']` 显示所有键

5. **权限判断结果**
   - `是否有 道法 权限: True/False` 显示每个学科的权限

# 学科管理系统 - 功能文档

## ✅ 功能概述

学科管理系统允许超级管理员管理学校的学科设置，并为教师分配任教学科。每个教师可以任教一个或多个学科，包括正副班主任、行政、校级领导等所有角色。

## 📋 功能特性

### 1. 学科管理
- ✅ 添加新学科
- ✅ 编辑学科信息
- ✅ 删除学科（需确保无教师任教）
- ✅ 查看学科列表
- ✅ 查看每个学科的任教教师数量

### 2. 任教分配管理
- ✅ 为教师分配学科
- ✅ 移除教师的任教学科
- ✅ 查看教师任教的所有学科
- ✅ 查看某学科的所有任教教师
- ✅ 按教师或学科筛选

### 3. 默认学科

- 语文

## 🗄️ 数据库设计

### subjects 表（学科表）
```sql
CREATE TABLE subjects (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,
    description TEXT,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    updated_at TEXT DEFAULT CURRENT_TIMESTAMP
);
```

### teacher_subjects 表（教师任教关联表）
```sql
CREATE TABLE teacher_subjects (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    teacher_id TEXT NOT NULL,
    subject_id INTEGER NOT NULL,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (subject_id) REFERENCES subjects(id) ON DELETE CASCADE,
    UNIQUE(teacher_id, subject_id)
);
```

## 🔌 API接口

### 学科管理API

#### 1. 获取所有学科
```
GET /api/subjects
```
**响应示例**:
```json
{
    "status": "ok",
    "subjects": [
        {
            "id": 1,
            "name": "语文",
            "description": "语文学科",
            "created_at": "2026-01-17 10:00:00",
            "updated_at": "2026-01-17 10:00:00"
        }
    ]
}
```

#### 2. 添加学科
```
POST /api/subjects
Content-Type: application/json

{
    "name": "学科名称",
    "description": "学科描述"
}
```

#### 3. 更新学科
```
PUT /api/subjects/{subject_id}
Content-Type: application/json

{
    "name": "新学科名称",
    "description": "新描述"
}
```

#### 4. 删除学科
```
DELETE /api/subjects/{subject_id}
```

### 任教管理API

#### 5. 获取教师任教的学科
```
GET /api/teacher-subjects/{teacher_id}
```

#### 6. 为教师分配学科
```
POST /api/teacher-subjects
Content-Type: application/json

{
    "teacher_id": "3",
    "subject_id": 1
}
```

#### 7. 移除教师的任教学科
```
DELETE /api/teacher-subjects/{teacher_id}/{subject_id}
```

#### 8. 获取学科的所有任教教师
```
GET /api/subject-teachers/{subject_id}
```

## 🎨 界面说明

### 访问路径
登录后，超级管理员可以在左侧导航栏看到"学科管理"菜单项。

### 学科列表标签页
- 显示所有学科
- 显示每个学科的任教教师数量
- 可以添加、编辑、删除学科
- 删除学科前会检查是否有教师任教

### 任教分配标签页
- 显示所有教师及其任教学科
- 可以按教师或学科筛选
- 可以为教师添加或移除学科
- 显示教师的角色和班级信息

## 📝 使用指南

### 如何添加学科

1. 登录超级管理员账号
2. 点击左侧导航栏的"学科管理"
3. 在"学科列表"标签页，点击"添加学科"按钮
4. 填写学科名称和描述
5. 点击"保存"

### 如何为教师分配学科

**方法1：从任教分配标签页**
1. 切换到"任教分配"标签页
2. 点击"分配学科"按钮
3. 选择教师和学科
4. 点击"保存"

**方法2：快速分配**
1. 在"任教分配"标签页找到目标教师
2. 点击该教师行的"添加学科"按钮
3. 选择学科
4. 点击"保存"

### 如何移除教师的任教学科

1. 在"任教分配"标签页找到目标教师
2. 在该教师的任教学科列表中，点击学科徽章上的"×"按钮
3. 确认删除

### 如何删除学科

1. 在"学科列表"标签页找到要删除的学科
2. 确保该学科没有教师任教（任教教师数为0）
3. 点击"删除"按钮
4. 确认删除

## ⚠️ 注意事项

### 权限要求
- 只有超级管理员可以管理学科和分配任教
- 普通教师无法访问学科管理功能

### 数据约束
- 学科名称必须唯一
- 一个教师不能重复任教同一学科
- 删除学科前必须先移除所有任教教师
- 删除教师时会自动删除其所有任教记录（级联删除）

### 角色说明
- 所有角色的教师都可以任教学科
- 正副班主任可以同时任教多个学科
- 行政和校级领导也可以任教学科
- 超级管理员通常不任教学科

🔄 绩效考核系统 - 自动刷新功能测试

📋 功能说明

绩效考核页面现在支持自动刷新功能，当后台管理员修改用户角色后，评分页面会自动更新教师列表。

✨ 核心特性

1. 自动定时刷新
每30秒自动检查一次数据变化
2. 手动刷新按钮
点击右上角"刷新"按钮立即更新
3. 智能暂停/恢复
切换标签页或失去焦点时自动暂停
4. 数据变化检测
只在数据真正变化时更新界面

⚙️ 配置选项

修改刷新频率：

编辑 js/performance.js 文件，找到：

}, 30000); // 30秒

修改为你想要的时间（毫秒）：

}, 60000); // 60秒（1分钟）
}, 15000); // 15秒
}, 10000); // 10秒

建议值：

  • 10秒：频繁更新，适合测试
  • 30秒：平衡性能和实时性（默认）
  • 60秒：节省资源，适合稳定环境

📊 性能优化

  • 智能检测：只在数据真正变化时更新界面
  • 资源节省：切换标签页时自动停止刷新
  • 网络优化：使用轻量级的数据哈希比较
  • 用户体验：静默刷新，不打断用户操作

🎯 使用场景

场景1：管理员修改用户角色

  1. 管理员在后台管理页面修改某个用户的角色
  2. 保存修改
  3. 30秒内，绩效考核页面自动更新
  4. 无需手动刷新浏览器

  1. 管理员在后台添加新用户，角色设为"正班主任"

  3. 30秒内，新教师出现在列表中

场景3：需要立即查看更新

  1. 管理员修改了用户角色
  2. 不想等待30秒
  3. 点击右上角的"🔄 刷新"按钮
  4. 数据立即更新

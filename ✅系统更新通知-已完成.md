## 🚀 使用方法

### 快速集成（2步）

**步骤1：** 在页面 `<head>` 中添加CSS
```html
<link rel="stylesheet" href="/css/update-notification.css">
```

**步骤2：** 在页面底部添加JS（在Bootstrap之后）
```html
<script src="/js/update-notification.js"></script>
```

### 测试方法

**方法1：** 访问测试页面
```
http://localhost:5000/test_update_notification.html
```

**方法2：** 清除localStorage后刷新
```javascript
localStorage.removeItem('system_last_seen_version');
```

**方法3：** 手动触发
```javascript
showUpdateModal();
```

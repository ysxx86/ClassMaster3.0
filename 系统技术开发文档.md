## 1. 系统概述

### 1.1 系统简介
ClassMaster 2.0是一个面向中小学的智能班级管理系统，主要服务于班主任、科任教师和学校管理员。系统集成了学生信息管理、成绩记录、AI评语生成、德育评价、报告导出等核心功能。

### 1.2 核心功能
- **用户权限管理**：支持超级管理员、正班主任、副班主任、科任老师、行政、校级领导六种角色
- **学生信息管理**：完整的学生档案管理，支持批量导入导出
- **成绩管理**：支持多学科成绩录入、智能导入、成绩分析
- **评语管理**：集成DeepSeek AI，智能生成个性化学生评语
- **德育评价**：六维度德育素质评价系统（品质、学习、健康、审美、实践、生活）
- **成绩分析**：多维度成绩统计分析、图表展示
- **班主任考核**：基于多项指标的班主任绩效考核系统
- **报告导出**：生成PDF/Word格式的学生综合素质报告
- **教师分配**：灵活的教师-班级-学科分配管理

### 1.3 技术特点
- 前后端分离架构，使用iframe实现页面模块化
- RESTful API设计，JSON数据交互
- 基于角色的权限控制（RBAC）
- 支持实时数据同步和更新通知
- 响应式UI设计，支持多种屏幕尺寸

## 3. 数据库设计

### 3.1 数据库概述
系统使用SQLite作为数据库，数据库文件名为`students.db`。所有表都启用了外键约束（`PRAGMA foreign_keys = ON`）。

### 3.2 核心数据表

#### 3.2.1 users表（用户表）
```sql
CREATE TABLE users (
    id TEXT PRIMARY KEY,                    -- 用户ID
    username TEXT NOT NULL UNIQUE,          -- 用户名（唯一）
    password_hash TEXT NOT NULL,            -- 密码哈希
    is_admin INTEGER NOT NULL DEFAULT 0,    -- 是否超级管理员（0/1）
    class_id TEXT,                          -- 关联班级ID
    primary_role TEXT DEFAULT '科任老师',   -- 主要角色
    reset_password TEXT,                    -- 重置密码（明文，用于找回）
    created_at TEXT,                        -- 创建时间
    updated_at TEXT                         -- 更新时间
);
```

**字段说明：**
- `id`: 用户唯一标识，字符串类型
- `username`: 登录用户名，必须唯一
- `password_hash`: 使用Werkzeug加密的密码哈希
- `is_admin`: 1表示超级管理员，0表示普通用户
- `class_id`: 正班主任关联的班级ID（外键关联classes表）
- `primary_role`: 用户角色，可选值：正班主任、副班主任、科任老师、行政、校级领导、超级管理员
- `reset_password`: 存储明文密码用于密码找回（仅用于演示，生产环境应移除）

#### 3.2.2 classes表（班级表）
```sql
CREATE TABLE classes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,   -- 班级ID
    class_name TEXT NOT NULL UNIQUE,        -- 班级名称（唯一）
    created_at TEXT,                        -- 创建时间
    updated_at TEXT                         -- 更新时间
);
```

**字段说明：**
- `id`: 班级唯一标识，自增整数
- `class_name`: 班级名称，如"二年级4班"，必须唯一
- 支持的班级名称格式：
  - 标准格式：二年级4班
  - 简写格式：二年4班
  - 数字格式：204（2表示二年级，04表示4班）
  - 年份格式：2024级4班（自动计算年级）

#### 3.2.3 students表（学生表）
```sql
CREATE TABLE students (
    id TEXT NOT NULL,                       -- 学号
    name TEXT NOT NULL,                     -- 姓名
    gender TEXT NOT NULL,                   -- 性别（男/女）
    class_id INTEGER NOT NULL,              -- 班级ID（外键）
    
    -- 体测数据
    height REAL,                            -- 身高(cm)
    weight REAL,                            -- 体重(kg)
    chest_circumference REAL,               -- 胸围(cm)
    vital_capacity REAL,                    -- 肺活量(ml)
    dental_caries TEXT,                     -- 龋齿情况
    vision_left REAL,                       -- 左眼视力
    vision_right REAL,                      -- 右眼视力
    physical_test_status TEXT,              -- 体测情况
    
    -- 学科成绩（等级制：优/良/及格/待及格）
    yuwen TEXT,                             -- 语文
    shuxue TEXT,                            -- 数学
    yingyu TEXT,                            -- 英语
    daof TEXT,                              -- 道法
    kexue TEXT,                             -- 科学
    zonghe TEXT,                            -- 综合
    tiyu TEXT,                              -- 体育
    yinyue TEXT,                            -- 音乐
    meishu TEXT,                            -- 美术
    laodong TEXT,                           -- 劳动
    xinxi TEXT,                             -- 信息
    shufa TEXT,                             -- 书法
    xinli TEXT,                             -- 心理
    
    -- 德育维度（分数制，总分100）
    pinzhi INTEGER,                         -- 品质（0-30分）
    xuexi INTEGER,                          -- 学习（0-20分）
    jiankang INTEGER,                       -- 健康（0-20分）
    shenmei INTEGER,                        -- 审美（0-10分）
    shijian INTEGER,                        -- 实践（0-10分）
    shenghuo INTEGER,                       -- 生活（0-10分）
    
    -- 评语
    comments TEXT,                          -- 学生评语
    
    -- 时间戳
    created_at TEXT,                        -- 创建时间
    updated_at TEXT,                        -- 更新时间
    
    PRIMARY KEY (id, class_id),             -- 复合主键
    FOREIGN KEY (class_id) REFERENCES classes(id) ON DELETE CASCADE
);
```

**重要说明：**
- 使用复合主键`(id, class_id)`，允许不同班级有相同学号的学生
- 学科成绩使用等级制，有效值：优、良、及格、待及格、/（未评）
- 德育维度使用分数制，各维度有分数上限，总分不超过100
- 所有数值字段允许NULL，表示未录入

#### 3.2.4 teaching_assignments表（教师任教分配表）
```sql
CREATE TABLE teaching_assignments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,   -- 分配ID
    user_id TEXT NOT NULL,                  -- 教师ID（外键）
    class_id INTEGER NOT NULL,              -- 班级ID（外键）
    subject_name TEXT NOT NULL,             -- 学科名称
    created_at TEXT,                        -- 创建时间
    updated_at TEXT,                        -- 更新时间
    
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (class_id) REFERENCES classes(id) ON DELETE CASCADE,
    UNIQUE(user_id, class_id, subject_name) -- 防止重复分配
);
```

**字段说明：**
- 记录教师在哪个班级教授哪门学科
- 一个教师可以在多个班级教授多门学科
- 用于权限控制：教师只能编辑自己任教班级和学科的成绩

def require_admin(f):
    """要求超级管理员权限"""
    @wraps(f)

class RealtimeSync {
    constructor() {
        this.lastUpdate = Date.now();
        this.checkInterval = 5000;  // 5秒检查一次
        this.listeners = {};
    }
    
    // 注册数据变更监听器
    on(dataType, callback) {
        if (!this.listeners[dataType]) {
            this.listeners[dataType] = [];
        }
        this.listeners[dataType].push(callback);
    }
    
    // 开始轮询检查更新
    start() {
        setInterval(() => {
            this.checkUpdates();
        }, this.checkInterval);
    }
    
    // 检查数据更新
    async checkUpdates() {
        try {
            const response = await api.get(`/api/realtime/check-updates?since=${this.lastUpdate}`);
            
            if (response.status === 'ok' && response.updates.length > 0) {
                response.updates.forEach(update => {
                    this.notifyListeners(update.type, update.data);
                });
                
                this.lastUpdate = Date.now();
            }
        } catch (error) {
            console.error('检查更新失败:', error);
        }
    }
    
    // 通知监听器
    notifyListeners(dataType, data) {
        if (this.listeners[dataType]) {
            this.listeners[dataType].forEach(callback => {
                callback(data);
            });
        }
    }
}

class VirtualScroller {
    constructor(container, itemHeight, renderItem) {
        this.container = container;
        this.itemHeight = itemHeight;
        this.renderItem = renderItem;
        this.data = [];
        this.visibleStart = 0;
        this.visibleEnd = 0;
        
        this.init();
    }
    
    init() {
        this.container.style.position = 'relative';
        this.container.style.overflow = 'auto';
        
        this.viewport = document.createElement('div');
        this.viewport.style.position = 'relative';
        this.container.appendChild(this.viewport);
        
        this.container.addEventListener('scroll', () => {
            this.update();
        });
    }
    
    setData(data) {
        this.data = data;
        this.viewport.style.height = `${data.length * this.itemHeight}px`;
        this.update();
    }
    
    update() {
        const scrollTop = this.container.scrollTop;
        const containerHeight = this.container.clientHeight;
        
        this.visibleStart = Math.floor(scrollTop / this.itemHeight);
        this.visibleEnd = Math.ceil((scrollTop + containerHeight) / this.itemHeight);
        
        // 渲染可见项
        this.render();
    }
    
    render() {
        this.viewport.innerHTML = '';
        
        for (let i = this.visibleStart; i < this.visibleEnd && i < this.data.length; i++) {
            const item = this.renderItem(this.data[i], i);
            item.style.position = 'absolute';
            item.style.top = `${i * this.itemHeight}px`;
            item.style.width = '100%';
            this.viewport.appendChild(item);
        }
    }
}

// 防抖函数
function debounce(func, wait) {
    let timeout;
    return function(...args) {
        clearTimeout(timeout);
        timeout = setTimeout(() => func.apply(this, args), wait);
    };
}

// 节流函数
function throttle(func, limit) {
    let inThrottle;
    return function(...args) {
        if (!inThrottle) {
            func.apply(this, args);
            inThrottle = true;
            setTimeout(() => inThrottle = false, limit);
        }
    };
}

function validateDeyuScore(dimension, score) {
    const max = DEYU_DIMENSIONS[dimension].max;
    
    if (score < 0) {
        return { valid: false, message: '分数不能为负' };
    }
    
    if (score > max) {
        return { valid: false, message: `${DEYU_DIMENSIONS[dimension].name}分数不能超过${max}` };
    }
    
    return { valid: true };
}

function validateDeyuTotal(scores) {
    const total = Object.values(scores).reduce((sum, score) => sum + (score || 0), 0);
    
    if (total > DEYU_TOTAL_MAX) {
        return { valid: false, message: `总分不能超过${DEYU_TOTAL_MAX}，当前总分：${total}` };
    }
    
    return { valid: true };
}

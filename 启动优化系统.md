## 常见问题

### Q1: 页面还是很慢?
**A**: 
1. 清除浏览器缓存 (Ctrl+Shift+Delete)
2. 检查Network标签,看是否还有iframe同时加载
3. 运行`python optimize_database.py`

### Q2: 实时更新不生效?
**A**:
1. 打开浏览器控制台(F12)
2. 查看是否有错误信息
3. 确认`window.RealtimeSync`存在
4. 确认已注册监听器

### Q3: 如何清除API缓存?
**A**:
```javascript
// 在浏览器控制台执行
PerformanceOptimizer.apiCache.clear();
```

### Q4: 如何监控性能?
**A**:
```javascript
const monitor = new PerformanceOptimizer.PerformanceMonitor();
monitor.start('操作名称');
// 执行操作
monitor.end('操作名称'); // 输出耗时
```

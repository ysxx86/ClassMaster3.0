## ⚠️ 如果遇到问题

### 问题1: 还是有404错误

**检查**:
```bash
# 确认文件存在
ls -la realtime_api.py

# 确认server.py修改正确
grep "realtime_bp" server.py
```

**解决**:
重新运行验证脚本:
```bash
python verify_optimization.py
```

### 问题2: 导入错误

**错误信息**:
```
ImportError: cannot import name 'realtime_bp' from 'realtime_api'
```

**解决**:
确认 `realtime_api.py` 在项目根目录,与 `server.py` 同级

### 问题3: 页面还是慢

**检查**:
1. 清除浏览器缓存 (Ctrl+Shift+Delete)
2. 确认iframe使用的是 `data-src` 而不是 `src`
3. 查看Network标签,确认资源按需加载

## 问题2：1年级2班没有显示

### 原因
1年级2班（class_id=57）没有学生数据，所以不会显示在成绩管理中。

### 数据验证
```bash
sqlite3 students.db "SELECT class_id, COUNT(*) FROM students WHERE class_id IN (56,57,58,59) GROUP BY class_id;"
# 结果：
# 56|37  (一年级1班)
# 58|38  (一年级3班)
# 59|37  (一年级4班)
# 注意：57 (一年级2班) 没有数据
```

### 解决方案
修改了成绩管理API，返回空班级信息：

**后端 (grades.py)**：
- 检测没有学生的班级
- 返回 `empty_classes` 数组

**前端 (js/grades.js)**：
- 显示空班级提示信息
- 例如："提示：一年级2班 暂无学生数据"

### 现在的行为
- 成绩管理会显示有学生的班级（1班、3班、4班）
- 页面顶部会显示提示："提示：一年级2班 暂无学生数据"
- 教师知道自己任教4个班级，但2班暂时没有学生

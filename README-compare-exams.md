# 多次考试成绩对比分析功能

## 功能概述

多次考试成绩对比分析功能允许教师选择多次考试进行对比分析，深入了解学生成绩的变化趋势和分布情况。主要功能包括：

1. 可选择多个考试进行对比分析
2. 通过图表直观展示各次考试的成绩统计
3. 分析学生个人成绩变化趋势
4. 识别进步显著和需要关注的学生

## 技术实现

### 前端实现

1. **考试选择和数据获取**
   - 在考试列表中添加复选框，支持多选考试
   - 通过API调用获取选中考试的详细数据

2. **数据可视化**
   - 平均分对比图：展示不同考试同一学科的平均分变化
   - 及格率和优秀率对比：对比各次考试的及格率和优秀率
   - 雷达图：多维度展示各项指标（平均分、及格率、优秀率、最高分、最低分）
   - 分数段分布图：对比不同考试的成绩分布情况
   - 学生成绩变化分布图：统计学生成绩上升和下降的分布情况

3. **学生成绩对比分析**
   - 学生成绩对比表格：详细展示每个学生在不同考试中的成绩变化
   - 自动计算排名变化和分数变化率
   - 根据成绩变化幅度对学生进行分组展示

### 后端实现

1. **数据处理API**
   - `/api/exams/compare`：接收要对比的考试ID列表，返回对比分析所需的综合数据
   - 处理多个考试的成绩数据，计算统计指标
   - 查找所有考试共有的学科集合

2. **数据分析**
   - 计算各项统计指标：平均分、最高分、最低分、及格率、优秀率等
   - 分析分数段分布：0-59分、60-69分、70-79分、80-89分、90-100分
   - 识别学生成绩变化模式：进步显著、稳定、退步显著

## 用户界面

1. **考试选择界面**
   - 考试列表中添加复选框，支持多选
   - "对比分析"按钮，仅在选中至少两次考试时启用

2. **对比分析结果界面**
   - 参与对比的考试列表展示
   - 学科选择下拉框，用于选择要分析的学科
   - 多个图表区域，直观展示对比结果
   - 学生个人成绩对比表格，展示详细的学生成绩变化

## 核心算法

### 排名变化计算
根据学生在不同考试中的成绩计算排名变化，正值表示排名提升，负值表示排名下降。

### 进步/退步学生识别
使用四分位数统计方法，识别进步显著和退步显著的学生：
- 进步显著：分数变化大于第三四分位数（Q3）
- 退步显著：分数变化小于第一四分位数（Q1）

### 学生成绩变化分布
统计学生成绩变化分布情况，将变化分为六个区间：
- 下降10分以上
- 下降5-10分
- 下降0-5分
- 提高0-5分
- 提高5-10分
- 提高10分以上

## 使用方法

1. 在考试列表页面，选中至少两次要对比的考试
2. 点击"对比分析"按钮
3. 在对比分析页面，从下拉框选择要分析的学科
4. 查看各种图表和表格，分析学生成绩变化情况
5. 点击学生姓名可查看该学生的详细成绩信息

## 实现效果

通过该功能，教师可以：
1. 直观了解班级整体成绩的变化趋势
2. 识别进步显著的学生，给予鼓励
3. 发现需要关注的学生，及时干预
4. 分析教学效果，调整教学策略

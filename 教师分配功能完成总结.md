## 技术实现

### 前端技术
```
HTML5 Drag & Drop API
├── dragstart - 开始拖拽
├── dragend - 结束拖拽
├── dragover - 拖拽经过
├── dragleave - 拖拽离开
└── drop - 放下
```

### 数据流程
```
1. 用户拖拽教师卡片
   ↓
2. 放到角色框/班级框
   ↓
3. 调用API更新数据库
   ↓
4. 返回成功/失败
   ↓
5. 更新本地数据
   ↓
6. 重新渲染界面
   ↓
7. 显示提示消息
```

### API复用
- 使用现有的 `/api/users/<user_id>` PUT接口
- 只更新 `primary_role` 或 `class_id` 字段
- 无需创建新的API
